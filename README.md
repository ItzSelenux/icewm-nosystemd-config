# icewm-nosystemd-config
IceWM config file for nosystemd distros, using elogind

put ```preferences``` on ```~/.icewm``` and restart icewm

- for this configuration to work, you must have elogind installed


Artix:

```pacman -S elogind```

Devuan:

```apt install elogind```

Void:

```xbps-install elogind```
